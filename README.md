# OpenStuff

OpenStuff 是一个运行在 macOS 图形界面环境中的“老师-学生”伴侣软件项目。

当前阶段目标：
- 不写业务代码，先完成工程目录规划。
- 为每个核心目录写清职责与未来实现方向。
- 在 `docs/` 内维护整体方案与实现进展。

## 目录概览

- `apps/macos/`：桌面端 GUI 应用（未来入口）。
- `core/`：核心能力（采集、知识建模、调度、执行、存储）。
- `modules/`：三大工作模式（教学、辅助、学生）。
- `scripts/`：脚本与工具（知识解析、技能转换、自动化任务）。
- `config/`：配置模板与环境变量说明。
- `tests/`：测试策略与未来测试用例组织。
- `docs/`：项目方案、阶段计划与实现进展。
- `assets/`：UI 原型、图标、演示素材。
- `vendors/openclaw/`：OpenClaw 源码（main 分支 vendor）。
